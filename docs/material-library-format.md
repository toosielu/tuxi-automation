# 小红书爆款素材库提供格式

素材库需要，但不是第一天必须有。没有素材库时，系统会直接用用户输入 + 爆款结构 + AI 模型生成；有素材库后，生成质量会更稳定，更像你自己的方法论。

## 推荐交付方式

你可以任选一种给我：

1. `CSV`：适合批量爆款帖子和选题库。
2. `Markdown`：适合方法论、SOP、提示词、课程资料。
3. `JSON`：适合已经结构化的案例库。
4. 图片文件夹：适合封面参考图，后续接图片识别时使用。

## CSV 格式

建议文件路径：

```text
data/materials/xhs-viral-posts.csv
```

字段：

```csv
niche,postType,title,hook,content,cta,tags,coverText,whyItWorks,riskNote
AI,引流爆款贴,副业新手先别乱买提示词,很多人不是不会用AI，是一上来就写得太像说明书,正文内容...,想要模板可以留言测试,#AI提示词 #小红书运营,副业新手先别乱买提示词,先别句式有避坑感，适合点击,不承诺收益
```

字段说明：

| 字段 | 说明 |
| --- | --- |
| niche | 赛道，比如 AI、商业、教辅、母婴 |
| postType | 引流爆款贴、干货贴、信任成交贴 |
| title | 小红书标题 |
| hook | 开头钩子 |
| content | 正文 |
| cta | 结尾引导 |
| tags | 标签，用空格分隔 |
| coverText | 封面主文案 |
| whyItWorks | 为什么有效 |
| riskNote | 合规风险提醒 |

## Markdown 格式

建议文件路径：

```text
data/materials/xhs-methods.md
```

示例：

```markdown
# AI 赛道爆款结构

## 痛点避坑型

- 适合：引流涨粉、私信咨询
- 标题公式：新手先别 + 错误动作
- 开头：很多人不是不会做，而是第一步就做重了
- 正文结构：错误做法 -> 替代动作 -> 小测试 -> 轻 CTA
- 风险：不要承诺收益，不要诱导站外
```

## 图片素材

建议目录：

```text
data/materials/covers/
```

命名方式：

```text
AI-避坑型-001.png
母婴-清单型-001.png
教辅-干货型-001.png
```

## 第一批素材建议

先不用很多，建议你先给：

- 每个赛道 10 条爆款标题
- 每个赛道 5 条完整帖子
- 每个赛道 5 张封面参考
- 你觉得成交效果好的私信话术 20 条
- 你自己的课程 SOP 或陪跑方法论 1 份

最优先赛道：

1. AI
2. 商业副业
3. 教辅资料
4. 母婴资料
5. 虚拟电商资料
